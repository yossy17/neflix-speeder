このスクリプトは、以下のサービスで再生速度を `0.25x` ~ `16x` から調整できる機能を提供します。

- [Prime Video](https://amazon.com/gp/video/storefront)
- [Netflix](https://netflix.com)

## 機能

| Key | Speed |
| :-: | ----: |
|  ;  | 0.25- |
|  '  | 0.25+ |
|  [  |  0.5- |
|  ]  |  0.5+ |
|  -  |    1- |
|  =  |    1+ |
|  `  |  0.5x |
|  1  |    1x |
|  2  |    2x |
|  3  |    3x |
|  4  |    4x |
