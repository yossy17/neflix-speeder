このスクリプトは、以下のサービスで再生速度を `0.25x` ~ `16x` から調整できる機能を提供します。

- [Prime Video](https://amazon.com/gp/video/storefront)
- [Netflix](https://netflix.com)

## 機能

| Key | Speed |      Supported       |
| :-: | ----: | :------------------: |
|  ;  | 0.25- | Netflix, Prime Video |
|  '  | 0.25+ | Netflix, Prime Video |
|  [  |  0.5- |       Netflix        |
|  ]  |  0.5+ |       Netflix        |
|  -  |    1- | Netflix, Prime Video |
|  +  |    1+ | Netflix, Prime Video |
|  `  |  0.5x |       Netflix        |
|  1  |    1x |       Netflix        |
|  2  |    2x |       Netflix        |
|  3  |    3x |       Netflix        |
|  4  |    4x |       Netflix        |
